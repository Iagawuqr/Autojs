SOBRE O SCRIPT

Script avançado para Roblox com contagem automática de números (0-5000) e sistema de pulo automático.

Pulo Automático UI Moderna Chat V1/V2 Configuração

ATENÇÃO

Este código é protegido contra downloads não autorizados. Apenas execução direta no Roblox é permitida.

COMO USAR

1

Copie o código abaixo

loadstring(game:HttpGet("https://seusite.rf.gd/raw.js"))()
2

Cole no seu executor

Execute no Roblox com qualquer executor compatível (PC ou Android).

3

Configure e use

Defina os números inicial/final, velocidade e clique em INICIAR.
